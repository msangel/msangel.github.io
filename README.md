## running locally with the same env
Prerequirements:
* [ruby](https://rvm.io/)
* [Bundler](https://bundler.io/) 

 1. Create `Gemfile`.
 2. run `bundle install` for installing all dependencies
 3. run locally with: `bundle exec jekyll serve`
 4. http://localhost:4000/ is your friend
